# check-ssh

Comprobar si hay conectividad con una lista de direcciones IP.

## Uso

Conectividad SSH (puerto 22)

```shell
check-ssh ips.txt [usuario]
```

Conectividad HTTP (puertos 80 y 443)

```shell
check-web ips.txt
```
